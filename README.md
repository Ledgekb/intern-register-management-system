
# Intern Register Management System

A full‑stack web application designed to replace a manual, paper‑based intern
registration and attendance process.

## 🚀 Features
- Intern registration and attendance tracking
- User authentication
- Reporting and data validation
- Admin and user workflows

## 🛠 Tech Stack
- Frontend: Angular
- Backend: Java, Spring Boot
- Database: MySQL / PostgreSQL
- Tools: Git, IntelliJ IDEA, Postman

## ⚙️ System Overview
The backend exposes RESTful APIs to manage interns, attendance records,
and reporting. The Angular frontend consumes these APIs to provide a
user‑friendly interface for administrators and users.

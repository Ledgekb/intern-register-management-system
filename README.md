
A full‑stack web application designed to replace a manual, paper‑based intern
registration and attendance process.

## 🚀 Features
- Intern registration and attendance tracking
- User authentication
- Reporting and data validation
- Role‑based access

## 🛠 Tech Stack
- Backend: Java, Spring Boot
- Frontend: Angular
- Database: MySQL / PostgreSQL
- Tools: Git, IntelliJ IDEA, Postman

## ⚙️ How It Works
The system exposes RESTful APIs for managing interns, attendance records,
and reports. The frontend consumes these APIs to provide a user‑friendly
interface for administrators and interns.
